# RobloxPy

# An API wrapper for the ROBLOX web api.

Examples:

```py 
import robloxpy
username = "ROBLOX"
groupid = "1"

print(robloxpy.Group.name(groupid))
print(robloxpy.User.is_online(username))
```
or..
```py
from robloxpy import Group, User
username = "ROBLOX"
groupid = "1"

print(Group.name(groupid))
print(User.is_online(username))
```
