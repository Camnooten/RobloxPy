# RobloxPy

# An API wrapper for the ROBLOX web api.

Examples:

```py 
import robloxpy
username = "ROBLOX"
groupid = "1"

print(robloxpy.Group.name(groupid))
print(robloxpy.User.is_online(username))
```
```py
import robloxpy

```
```py
import robloxpy
id = 50
print(robloxpy.Image.get_head_thumbnail(id))
```

import robloxpy

print(User.friend_count(id))
